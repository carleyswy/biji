# http状态码
## 状态码分类
- 1** 服务器收到请求
- 2** 请求成功
- 3** 重定向
- 4** 客户端错误
- 5** 服务端错误
&nbsp;

## 常见状态码
- 200 成功
- 301 永久重定向（配合location，浏览器自动处理）
- 302 临时重定向（配合location，浏览器自动处理）
- 304 资源未被修改
- 404 资源未找到
- 403 没有权限
- 500 服务器错误
- 504 网关超时
